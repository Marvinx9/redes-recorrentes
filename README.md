### Redes Neurais Recorrentes

#### Mapa do repositório

arquivo -> celula-recorrente-basica.ipynb
Se encontra o modelo mais simples de redes recorrentes, onde se utiliza (RNNCell) do pytorch para realizar uma previsão do país de origem mais provável de um sobrenome de uma pessoa.

arquivo -> rede-recorrente.ipynb
É baseado no arquivo celula-recorrente-basica.ipynb, com a diferença de utilizar aprendizagem profunda para processamento de dados sequenciais. Sua principal melhoria é na capacidade de manter um tipo de "memória" do contexto passado.
